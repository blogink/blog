Electron初识：将web页面变成桌面应用程序

前排提示：使用这东西需要有一定的耐心，配置出故障是正常的。本文虽不是面向小白，但自认为比较详细（恐但凡疏忽便再难成功）。

之前使用过一个名为 Nativefier 的工具，它基于 Electronjs，可以将网页打包为桌面客户端，颇有意思。玩久了感觉命令写来有点琐碎，故直接尝试 Electron。

结果耗时超出我的预料，顺带输了不该输的东西再次把本地环境弄乱。尝试良久，一路坎坷，终于打包成功。

以防以后踩坑再去搜索半天，故于此记录。

本文不讨论 Electron 技术的缺点。

说明

  • 若无说明，本文描述及举例均为 Windows 平台下操作；
  • 本文仅为初级用法，完整使用请看 官网文档；
  • 为行文不过于突兀，可能存在“多余的”文字。建议全部看完再操作，不建议看一步做一步；
  • 千万不要多打字或少打字，打错字则更是不可。

介绍

Electron 是一个使用 JavaScript、HTML 和 CSS 构建桌面应用程序的框架。 嵌入 Chromium 和 Node.js 到 二进制的 Electron 允许您保持一个 JavaScript 代码代码库并创建 在 Windows 上运行的跨平台应用 macOS 和 Linux。

只看初级功能，你可以简单粗暴地把它理解为以桌面客户端的方式直接访问 Web 页面。
