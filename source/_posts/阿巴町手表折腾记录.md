<font color="red"><b>另外，折腾有风险，玩机需谨慎。本教程仅作参考，出现任何后果本站不负责任。</b></font>

最近找到了曾经使用过的一款阿巴町V118手表。虽然现在已经使用手机不需要这个了，但从网上得知了这玩意基于安卓6.0，再加上看到很多人将其破解，便萌生了动手的想法。
虽然配置1G+8G，但也不是不能用~

# 材料
- 阿巴町V118手表（请确认其基于安卓系统而非AliOS等）
- 电话卡（不管欠不欠费）
- 电脑端安卓工具（例如搞机助手等，本文以此为例）
- 能用的安卓桌面（例如NovaLauncher等）
- ADB环境（网上有教）
- 配套数据线

# 过程
## 进入原生系统设置
先给手表把电话卡装上（也可以先充电再装卡，但要重启后才识别），之后充电至起码30%以上。然后在手表自带桌面的首页从下往上滑呼出拨号界面，输入`112#`，随后会弹出一个白色背景的拨号界面。

在这个界面中，删除前面输入的数字，输入`*#*#4636#*#*`，进入测试菜单。

在此界面点击`使用情况统计数据`一项后，点击左上角返回，进入原生系统设置。

## 手表端设置
【注：此处和其他正常安卓手机进入开发者模式方法相同】

划到最底下，点击`关于手机`，划到最底，点击`版本号`七次，打开开发者模式。

回到上级菜单，点击`开发者模式`条目，在里面把`USB调试`打开。其他你觉得有用的请自便。

再回到上级菜单，找到`无障碍`，将`快速开机`打开。

再回到上级菜单，进入`应用`，点击右上角三点，选择`显示系统进程`，找到`Launcher`和`KTWatchService`，分别点击，将其`停止运行`。完成上述步骤后将手表连接电脑

## 电脑操作
如果按照我的教程，可以使用`搞机助手`这个工具。保持手表在系统设置菜单界面，连接电脑，手表端提示授权点同意即可。待检测到设备后，点击`系统模式`选项卡，点击`软件卸载#冻结`

点击右侧`获取所有包名`，搜索找到下面这两项：
>com.cwtcn.kt.luncher
com.cwtcn.kt.watchservice
将其分别`冻结`，然后回到软件主界面找到`应用安装`，选中你先前准备好的安卓桌面apk文件，等待安装完成。
【不建议卸载，冻结即可。而且建议顺带将这两个备份一下】

## 启用桌面
在手表端设置界面找到`应用`进行寻找，确认下你刚才选择的桌面已经安装完成（在列表显示）。然后回到上级，点击`主屏幕`，选中你准备的桌面。

不出意外的话桌面能够正常启用了。

# 换回原生桌面？
由于每次重启后设置里的USB调试会自动关闭，因此需手动打开后将手表连接电脑。按照先前冻结应用的方法，将那两个应用解冻，完成后重启手表即可。

（这也是为什么网上很多教程都建议修改apk包名的原因，重启手表后，原生桌面和原生手表服务组件会清理非`com.cwtcn.kt.XXX`的应用）

# 推荐软件
<font color="red"><b>警告：这些软件别用在主力机上。这里面有些软件各位可能不屑一顾，但对这种手表来说已经算可以凑合了。你有更佳的软件请随意。</b></font>

| 类型 | 软件名 | 提醒 |
| ------------ | ------------ | ------------ |
| 启动器 | NovaLauncher | 此无需过多介绍，自定义桌面比较出名的，不知道的自行百度。呼出顶栏要么黑屏（再次上滑恢复，可理解为状态栏和下拉菜单是黑屏）要么没反应，暂时难以解决，估计是阿巴町官方修改了安卓系统文件 |
| 虚拟按键 | easyTouch | 阿巴町官方肯定不会是只在安卓系统上单纯开发了套软件，因为即使替换了原生桌面，这种通知栏、虚拟导航什么的还是难以调出，故使用此软件“曲线救国”，类似于iOS的小白点 |
| 输入法 | QQ输入法 | 阿巴町输入法给它自己那套软件用用还凑合的。安装后分辨率可能让人很难受，但不至于不能用 |
| 应用市场 | 2345应用市场 | 实在找不出占用小且功能全的这种配置上的应用市场了。不建议使用新版的，建议版本为下载完后图标整体浅蓝、顶部与手提袋绳为深蓝，没有或章鱼图标不明显 |
| 浏览器 | Via | 内核版本有点低，有些网站加载不来或加载故障 |
| 文件管理 | Mi File Explorer | 从小米手机中提取移植的，功能不错 |
